# Nobel Civils Website

A responsive business website built for a civil engineering company as part of a client-based development task.

## Features

- Responsive design
- Service showcase
- Project portfolio section
- Contact form (frontend)
- Real company media integration

## Tech Stack

- HTML
- CSS
- JavaScript

## Purpose

This project was built to demonstrate the ability to take a real client brief and transform it into a functional and professional website.

## Future Improvements

- Backend integration for contact form
- CMS for updating projects
